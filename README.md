# Power & Social Stratification Quest

A one-page classroom game about power, social stratification, institutions, inequality, and life chances. The game is built as a static HTML file, so it can be hosted publicly on GitHub Pages without any build tools.

## Files you need in a new GitHub repository

- `index.html` — the playable game page.
- `README.md` — these setup instructions.

## Publish it with GitHub Pages

1. Create a new public repository on GitHub.
2. Upload `index.html` and `README.md` to the root of the repository.
3. Open the repository on GitHub.
4. Go to **Settings**.
5. In the left menu, click **Pages**.
6. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
7. Under **Branch**, choose `main` and `/ (root)`, then click **Save**.
8. Wait a minute or two for GitHub to publish the site.
9. Your public game link will look like:

   `https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

## Example

If your GitHub username is `ms-smith` and your repository is named `power-game`, the public link would be:

`https://ms-smith.github.io/power-game/`

## Updating the game later

Edit `index.html`, commit the change to the `main` branch, and GitHub Pages will automatically republish the updated version.
